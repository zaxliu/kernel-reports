GPU Kernel

GPU 上运行的计算函数。深度学习中的矩阵乘法、归一化、注意力等算子最终都编译为 kernel。 手写高性能 CUDA kernel 需要极深的硬件知识。

Agent 循环

受 Karpathy autoresearch 启发：LLM 自主修改代码 → 运行 benchmark → 保留改进 / 回退失败 → 重复。 无需人工干预，一夜可跑 100+ 次迭代。

KernelBench

斯坦福发布的算子优化评测集。4 个难度级别：L1 单算子、L2 算子融合、L3 模型级、L4 HuggingFace 模型。 每题给出参考实现，要求生成更快且数值正确的 kernel。
