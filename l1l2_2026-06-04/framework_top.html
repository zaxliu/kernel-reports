§2 头部 Winners 解法

§2.1 ako4all_L2_P56 — Linear → Sigmoid → Sum, 6.25× verified

题型: Linear(32768→32768) + Sigmoid + Sum,batch=128。

优化路径 (11 个 iter):

```

Iter 0 Baseline 1.01×

Iter 1 FP16 tensor core matmul 3.35×

Iter 7 INT8 per-channel weight quant 6.27× ← 跨过 FP16 上限

Iter 9 INT8 + torch.compile postproc 6.59× (claim max)

```

核心技术: torch._int_mm (INT8 tensor core) + per-channel weight quant + torch.compile postproc fusion。

```python

self.inv_scale_h = (w_abs_max / 127.0).t().contiguous().to(torch.float16)

self.weight_i8 = (w / w_abs_max * 127).round().to(torch.int8)

out_i32 = torch._int_mm(x_i8, self.weight_i8.t())

@torch.compile(fullgraph=True, dynamic=False, mode="reduce-overhead")

def _postprocess(out_i32, x_scale, inv_scale_h, bias_h):

out = out_i32.to(torch.float32)

out = out (x_scale inv_scale_h)

return torch.sum((out + bias_h).sigmoid_(), dim=1, keepdim=True)

```

三个独立框架在同一题(minGPT GELU)上都跑到 ~8.87×,verified ≈ claim。这是公认稳定可优化的题目,各框架 LLM 都找到了正确路径。

ako4all_L1_P88 的 kernel(纯 fp32 Triton GELU,无 cast):

```python

@triton.jit

def gelu_kernel(x_ptr, out_ptr, n_elements, BLOCK_SIZE: tl.constexpr):

pid = tl.program_id(0)

block_start = pid * BLOCK_SIZE

offsets = block_start + tl.arange(0, BLOCK_SIZE)

mask = offsets < n_elements

x = tl.load(x_ptr + offsets, mask=mask)

coeff = tl.extra.cuda.libdevice.sqrt(2.0 / math.pi)

x3 = x x x

inner = coeff (x + 0.044715 x3)

tanh_val = tl.extra.cuda.libdevice.tanh(inner)

result = 0.5 x (1.0 + tanh_val)

tl.store(out_ptr + offsets, result, mask=mask)

```
